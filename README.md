# AiPO

"AiPO" is a Python program for scraping data from [AiPO](https://aipo.myiqdii.com/).

## Installation

Use the package manager [git](https://git-scm.com/) to install "AiPO".

```bash
git clone https://github.com/zachkongoyu/AiPO.git
```

## Usage

```bash
python scrape.py
```
